Some services require you to get authentication tokens for scdl so that it can connect to your service accounts. This page helps you on the way.

SoundCloud

If you want to use scdl, you must first authenticate to access your SoundCloud account.

  1. it's fully javascript, no information shared with this site.

  2. Add the token to the config value in your scdl.conf file:

            auth_token = ...token goes here...

For more information see the scdl docs.
